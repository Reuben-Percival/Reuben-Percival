## Things I've built

**[rustpack](https://github.com/Reuben-Percival/rustpack)** — A package manager built on ALPM, the same library pacman uses. This one goes pretty deep into how Arch actually manages packages.

**[Melon](https://github.com/Reuben-Percival/Melon)** — An AUR helper written in Zig. Handles dependency resolution, PKGBUILD parsing, the works.

**[parut](https://github.com/Reuben-Percival/parut)** — A nicer frontend for paru. Because paru is great but the UX has room to breathe.

**[rustfetch](https://github.com/Reuben-Percival/rustfetch)** — System fetch tool in Rust, available on the AUR. The one that got the most traction.

**[zigfetch](https://github.com/Reuben-Percival/zigfetch)** — Rewrote rustfetch in Zig. Turned out better than expected.

**[rtimer](https://github.com/Reuben-Percival/rtimer)** — A terminal timer. Small and does exactly what it says.
